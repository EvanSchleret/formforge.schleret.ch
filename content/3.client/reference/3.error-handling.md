The client normalizes backend and network failures into a stable shape.

## Common fields

- `status`
- `code`
- `message`
- `fieldErrors`
- `businessCode`

## Typical handling

```ts
try {
  await management.createForm(input)
} catch {
  console.error(management.clientError.value)
}
```
