`FormForgeResponse` displays a read-only submission.

```vue
<FormForgeResponse
  :form-key="'contact'"
  :response-uuid="'submission-uuid'"
  layout="line"
/>
```

Main props:

- `response-uuid` (required)
- `form-key` (optional if inferred)
- `layout` (`line` or `column`)
