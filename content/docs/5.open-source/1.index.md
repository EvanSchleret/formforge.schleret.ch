FormForge is an open-source project built in public. The backend and client packages evolve together, but they can be installed and used independently.

The public API is usable for active projects. Review the supported Laravel and package versions in the repository before upgrading, especially while the project is moving toward its first stable release.

Use the [roadmap](/docs/open-source/roadmap) to see planned work and the [releases](/docs/open-source/releases) page to review published versions.

- [FormForge Client](https://github.com/EvanSchleret/FormForgeClient)
