This wiki exposes an MCP endpoint at:

- `https://formforge.schleret.ch/mcp`

Use this when you want assistants to:

- discover available pages
- read specific wiki chapters
- suggest an integration mode using a built-in prompt

## What is exposed

- Tools: `list-pages`, `get-page`
- Prompt: `suggest-integration-mode`
- Resource: `wiki-overview`
