## Overridable models

Config key: `formforge.models.*`

- `form_definition`
- `form_category`
- `form_submission`
- `submission_file`
- `staged_upload`
- `idempotency_key`
- `form_draft`
- `submission_automation_run`
- `submission_privacy_policy`
- `submission_privacy_override`

Rule: custom model classes must extend package base models.
