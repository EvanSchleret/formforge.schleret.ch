---
title: API Overview
description: Endpoint groups, prefixes, and high-level API surface.
---

Default prefix: `/api/formforge/v1`

Endpoint groups:

- `schema`
- `submission`
- `upload`
- `resolve`
- `draft`
- `management`

You can toggle groups in `formforge.http.endpoints`.

`resolve` now includes targeted field validation endpoints to validate one input without submitting a full payload.
