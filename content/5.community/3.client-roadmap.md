## Roadmap client

- [ ] Add dedicated `useAsyncData` composables for schema, management, and responses.
- [ ] Add typed server-side filters for forms listing (`category`, `status`, `search`, `published`).
- [ ] Add pagination primitives on list composables (`page`, `perPage`, `total`, `next`, `prev`).
- [ ] Add batch management actions (publish, unpublish, delete, category assign).
- [ ] Add optimistic UI updates for create/update/delete on forms and categories.
- [ ] Add renderer extensibility slots for field wrappers, error summaries, and custom actions.
- [ ] Add custom field type API for renderer and builder.
- [ ] Add autosave drafts with conflict detection and recovery.
- [ ] Add offline queue support for submissions and draft sync.
- [ ] Add built-in analytics hooks for submit/start/abandon events.
- [ ] Add response export helpers (CSV, JSON) with scoped permission checks.
- [ ] Add file upload progress and cancel support for `staged` and `managed` modes.
- [ ] Add i18n packs and runtime locale switching for renderer and builder labels.
- [ ] Add first-class examples for multi-tenant scoped routes and management-only scoped mode.
