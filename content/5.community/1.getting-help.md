FormForge is an open project built in public.

The community is pragmatic and builder-focused: feedback, bug reports, and implementation discussions are welcome.

## Where to ask

- Backend package issues: [github.com/EvanSchleret/FormForge/issues](https://github.com/EvanSchleret/FormForge/issues)
- Client package issues: [github.com/EvanSchleret/FormForgeClient/issues](https://github.com/EvanSchleret/FormForgeClient/issues)
- GitHub profile: [github.com/EvanSchleret](https://github.com/EvanSchleret)
- X / Twitter: [@EvanSchleret](https://twitter.com/EvanSchleret)

## Contribution mindset

- Reproducible bug reports are preferred.
- Small focused PRs are easier to review and merge.
- Docs and examples are first-class contributions.

## Current status

The community space is evolving. For now, collaboration happens primarily through GitHub and direct feedback channels.
